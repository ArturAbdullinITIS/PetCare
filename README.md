# Pet Care

## About

Pet Care is a mobile app for managing pet profiles, tracking care activities, viewing statistics, setting reminders, exploring public pet profiles, and playing a mini-game.

## Features

- Sign up and log in with email/password or Google
- Add and manage pet profiles
- Track activities such as:
  - Walks
  - Grooming
  - Vet visits
- Set reminders for grooming and vet appointments
- View pet analytics and activity statistics
- Explore public profiles of other pets
- Play a mini-game with your pet as the character
- Edit user profile and app settings
